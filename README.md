[English](./README_en.md) | [中文](./README.md)

面向对象实践 | 飞机大战安卓移植-后端

该脚本改自网上博客，链接在代码里有提供，只负责数据传输以及和客户端通信，游戏逻辑在安卓端。

> 因为学校定位「安卓移植」这种说法比较让人误解，导致游戏逻辑被迁移到了安卓前端。正常应该是服务器跑游戏逻辑，然后前端只负责渲染的，加上当时个人水平有限，前端写完了才意识到这个问题，但因为在期末，最后也就只能这样了。
